Kebijakan Privasi Bams Learning

Terakhir diperbarui: 4 Januari 2026

1. Informasi yang Kami Kumpulkan

Aplikasi Bams Learning mengumpulkan informasi berikut:

  • Data Akun: Nama, email, dan foto profil saat mendaftar
  • Data Pembelajaran: Progress belajar, nilai ujian, dan aktivitas dalam aplikasi
  • Data Perangkat: Token notifikasi untuk mengirim pemberitahuan
  • Kamera: Digunakan untuk scan QR code absensi dan foto profil
  • Lokasi: Digunakan untuk verifikasi absensi (opsional)

2. Penggunaan Informasi

Informasi yang dikumpulkan digunakan untuk:

  • Menyediakan layanan pembelajaran online
  • Mengirim notifikasi terkait pembelajaran
  • Meningkatkan kualitas aplikasi
  • Komunikasi antara guru dan siswa

3. Penyimpanan Data

Data disimpan dengan aman di server kami dan dilindungi dengan enkripsi. Kami tidak menjual atau membagikan data pribadi kepada pihak ketiga.

4. Hak Pengguna

Pengguna berhak untuk:

  • Mengakses data pribadi mereka
  • Meminta penghapusan akun dan data
  • Menonaktifkan notifikasi
